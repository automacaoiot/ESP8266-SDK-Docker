Bem-Vindo ao Ambiente de Desenvolvimento Ágil da Automação IOT. Este ambiente foi criado exclusivamente para facilitar o uso da SDK e suas configurações com o ESP8266. Na esquerda há um painel mostrando seus arquivos, é como se fosse sua Área de Trabalho. Essa WorkSpace está localizada na pasta compartilhada quando você inicializou esse Container no Docker. No diretório /workspace/iot/examples/ você irá encontrar diversos exemplos para testar e compreender o modo de uso da nossa SDK. Caso queira realizar uma compilação de teste, recomendadamos escolher o projeto **NOME DO PROJETO** para começar. Para fazer isso, abra o Terminal e digite nele: 
				  cd iot/examples/
				  make
Espere a compilação do make terminar para poder gravar seu programa no seu dispositivo ESP8266. Feito isso, confirme se o dispositivo está conectado na porta USB e digite: 
				make flash
Ao final deste processo, seu ESP8266 irá conter seu programa rodando nele e irá apresentar no Terminal seu Serial Output ( Saída Serial ), mostrando informações de Debug e outras informações do sistema. Feito isso, você concluiu seu primeiro projeto! Agora o resto é com você, **Happy Coding!**
